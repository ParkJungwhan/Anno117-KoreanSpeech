# Anno117-KoreanSpeech

### 유비소프트 Anno 177: 팍스 로마나 한국어 음성파일 만들기

- 한국어 음성 더빙이 없어서 영어 음성 파일을 기준으로 대사를 통해 한국어 음성파일을 만들어 처리하기위한 프로젝트

- 해당 프로그램은 mod가 아닌 아노117 리소스 파일을 재패키징 하는 부분이기 때문에 아노117 실행 프로그램에 의해 체크섬 해쉬값 판정시에 실패하여 진행이되지 않을수 있다. 해당 부분은 아래에서 다시 설명함
